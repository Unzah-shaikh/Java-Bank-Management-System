# 💳 Java Bank Management System

A **console-based Bank Management System** built using Java, designed to simulate real-world banking operations. This project demonstrates core programming concepts like user authentication, transaction handling, and data management using Java Collections.

---

## 🚀 Features

- 🔐 **User Registration & Login**
- 🏦 **Bank Account Creation**
- 💸 **Debit & Credit Operations**
- 📊 **Balance Inquiry**
- 🔄 **Money Transfer Between Accounts**
- 🧾 **Transaction History Tracking**
- 💼 **Loan Request System**
- 📞 **Customer Support Simulation**
- 👨‍💼 **View All Account Balances (Admin-like Feature)**

---

## 🛠️ Technologies Used

- **Java**
- **Java Collections Framework** (HashMap, ArrayList)
- **Console-Based Interface**

---

## 📌 How It Works

1. User registers with email and password.  
2. User logs in securely.  
3. Creates a bank account with an initial balance and PIN.  
4. Performs banking operations such as:
   - Deposit / Withdraw
   - Transfer money
   - Check balance  
5. System maintains transaction history for each user.

## ▶️ How to Run the Project

1. Clone the repository:
